Embed Example

TwiML scripts can be embedded in your site using an iframe. Just set src to the URL for your TwiML and append the ?embed=true. For example,

<iframe
  src="//quicktwiml.herokuapp.com/TwiML/YAK2vE__ecc?embed=true"
  width="100%"
  height="345px"
></iframe>
